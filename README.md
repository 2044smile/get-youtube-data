# google-api-python-client
Writing a blog with google api python client

* python 3.10
* google-api-python-client
* pandas

## Examples of use

```python
poetry run python get_test_data.py -c 곽튜브
# playlists
poetry run python get_test_data.py -c 곽튜브 -n 0
# playlistItems
```
